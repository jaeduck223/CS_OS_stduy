운영체제 : 컴퓨터 부품들을 실질적으로 동작시켜 주는 가장 근원적인 프로그램. 자원들을 프로그램에 할당하고 올바르게 실행되도록 돕는 특별한 프로그램
자원(Resource) : 프로그램 실행에 마땅히 필요한 요소들(CPU, 메모리, 입출력 장치, 파일 등)
메모리 영역 : 메모리는 크게 사용자가 실행하는 일반 응용 프로그램이 적재되는 사용자 영역과 운영체제가 적재되는 커널 영역으로 나뉘어짐.
운영체제의 역할 : cpu 관리, 메모리 관리, 입출력 장치 관리, 파일 시스템 관리 

운영체제를 공부해야 하는 이유 : 개발의 편리성, 문제 해결력 향상, 하드웨어의 제어 원리 이해 등등
