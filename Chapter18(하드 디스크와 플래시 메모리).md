RAM은 휘발성, 보조기억장치는 비휘발성의 특징을 가지는 저장소이다.
메모리에서 성능과 경제성 사이에서는 Trade-Off 관계가 발생합니다.

하드 디스크 : 자기적 방식을 사용하여 데이터를 저장하는 장치입니다. 데이터는 플래터 표면에 코팅된 자기 물질의 N극과 S극 방향을 통해 기록됩니다.
플래터(데이터가 저장되는 곳), 스핀들(플래터를 회전시키는 축), 헤드(자기 정보를 읽고 쓰는 바늘), 디스크 암(헤드를 받치는 지지대)으로 구성되어 있습니다.

데이터의 저장 단위 : 트랙, 섹터, 실린더
데이터가 물리적으로 저장되는 위치에 따라 CPU가 체감하는 속도는 많이 다르다.

플래시 메모리 : 전기적인 원리로 데이터를 관리하는 반도체 기반 저장장치. 주로 낸드 플래시가 사용됩니다.
셀 : 데이터를 저장하는 최소 단위. 셀에 몇 비트를 담느냐에 따라 성능과 수명이 좌우됩니다.
SLC -> 1 bit. MLC -> 2 bit. TLC -> 3 bit.
플래시 메모리는 덮어쓰기가 불가능합니다.
플래시 메모리는 읽기/쓰기와 삭제의 단위가 다르다. (페이지와 블록)
플래시 메모리의 구성 : 셀 -> 페이지 -> 블록 -> 플레인 -> 다이 (플레인과 다이에서 병렬성을 제공함.)

가비지 컬렉션 -> 추가적인 데이터 복사 및 블록 삭제 연산을 유발함.
