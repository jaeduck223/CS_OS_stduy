운영체제(Operating System) : 컴퓨터 환경을 구성하는 소프트웨어.
커널 : 운영체제에서 자원 관리와 프로그램 실행 제어라는 본질적 서비스를 담당하는 부분을 의미.
사용자 인터페이스(User Interface, UI) : 사용자와 컴퓨터 사이의 소통 경로

GUI(Graphic User Interface): 마우스와 그래픽으로 그려진 아이콘을 이용해서 사용자와 컴퓨터 간의 소통을 하도록 하는 것.
CLI(Command Line Interface): 텍스트 명령어를 통해 사용자와 시스템이 상호작용 하는 것.

이중 모드 : 응용 프로그램이 하드웨어 자원인 CPU, 메모리, 저장장치에 함부로 접근하지 못하도록 막아주는 기능.
사용자 모드 -> 허가되지 않은 접근이 발생하면, CPU가 예외를 발생시킴.
커널 모드 -> 모든 자원에 대한 접근이 가능한 모드.

시스템 콜(System call) : 응용 프로그램이 자원 접근을 필요로 하는 경우, 운영체제에 도움을 요청하는 방식 - 연산 오버헤드를 동반함.
