컴퓨터 시스템의 구성 요소 :  CPU, 메모리, 보조기억장치, 입출력장치

RAM(Random Access Memory) : 실행 중인 프로그램의 데이터와 명령어가 실시간으로 머무르는 주된 공간
전원이 차단되면 저장된 데이터가 즉시 소멸되는 특성이 있음.
CPU가 즉각적으로 처리해야 할 데이터를 대상으로 저장을 하면 되는 특성이 있다.
대용량 RAM은 CPU가 방대한 데이터에 즉각 접근하게 함으로써 시스템의 멀티태스킹 능력을 극대화하는 특성을 가집니다.

DRAM (Dynamic RAM) : 데이터가 시간에 따라 자연적으로 소멸하는 성질을 가집니다. 주기적으로 데이터를 다시 채워 넣어 줘야 합니다. 좁은 면적에 많은 데이터를 담을 수 있고 가격이 저렴하여 메인 메모리로 자주 쓰입니다.
SRAM (Static RAM) : 전원이 공급되는 동안에는 데이터가 소멸하지 않고 안정적으로 유지됩니다. 적은 용량으로도 최고의 속도를 내야 하는 캐시 메모리에 주로 사용됩니다.

SDRAM (Synchronous DRAM) : CPU의 클럭 신호에 맞춰 박자를 맞추며 데이터를 전송함. 클럭과 동기화됨으로써 CPU는 데이터가 언제 도착할지 예측할 수 있게 되어서 전송 효율이 상승하게 되었습니다.
SDR,DDR(Single, Double Data Rate) : 동일 클럭에서 한 주기당 데이터 하나를 보내는지 2개를 보내는지가 차이임.
