CPU 처리 능력을 결정하는 요소 : 클럭, 코어, 스레드(Thread)

클럭(Clock): 하드웨어 동기화와 속도의 임계값
클럭 신호는 컴퓨터 내부의 모든 부품이 정해진 박자에 맞춰 일사불란하게 협업할 수 있도록 조율하는 '지휘자'의 역할을 수행합니다. 
클럭 속도는 초당 반복 횟수를 의미하는 헤르츠(Hz) 단위로 측정됩니다. 

코어 : 명령어를 인출, 해석, 실행하는 실질적인 하드웨어 단위

하드웨어 스레드: 하나의 코어가 물리적으로 동시에 처리하는 명령어의 단위.
소프트웨어 스레드: 프로그램 내에서 독립적으로 실행되는 기능적 단위.

CPU가 여러 스레드를 극도로 빠르게 교차 실행하는 컨텍스트 스위칭을 통해 사용자에게는 모든 작업이 동시에 처리되는 것처럼 보이는 시각적 동시성을 제공함!
